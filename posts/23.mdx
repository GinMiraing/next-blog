十月即将过去，回顾这一个月的生活，似乎每天都是两点一线——家和公司来来回回。国庆后因为调休连上了七天班，把国庆放假的快乐全消磨完了。最近又有消息说今年除夕放不放假，全在公司的决定，总觉得在时代的发展中，人们的生活质量不增反降，更可怕的是，这种下降更多体现在精神生活上。

虽然每天都在抱怨上班太累，但班总归是要上的，那就只能在上班的时候多摸摸鱼，打发打发时间。最近我的工作量减少了，于是我在工作时间偷偷卷后端，虽然 Next 和 Remix 的官网都说它们是全栈框架，但我总觉得它们服务端能做的事有点少，也有可能是我的使用方法不对，这两个框架的服务端，我经常用来当作请求的中转站。由于只会 JavaScript 语言，所以入门后端我选的是 Nestjs 框架，目前看了两天，大致能写个后台。我觉得后端的难点并不是 CURD，这个谁都会写，真正的难点在于当数据量过于庞大时，如何以合适的方法返回数据；如何对请求进行鉴权，防止资源被盗刷；以及怎么进行分布式的部署等等等等......最近我在写一个评论系统，之后可能会出几篇 Coding 记录。

上周末厦门有一场漫展，我和女朋友一起去逛了，这还是我第一次去漫展。其实早在初中，我就步入了二次元的大门，只不过大学之后二次元的浓度降低了，也没怎么看新番。这次去漫展最大的体验就是：放眼望去一堆原神 Coser，夹杂着许多粉色波奇酱😂。感觉很多老番都过气了，没看到几个 Coser，当年我入坑的时候，还是刀剑神域、夏娜、凉宫春日这些番流行的年代。我女朋友 Cos 的星宫莓光看衣服还被人认成了小圆（其实我也觉得挺像的）。

趁着双十一快来了，我入手了一个键盘（之前的键盘 C 键坏了，怀疑是我经常 Ctrl + C 的原因🤡）——狼蛛 F98，打完折 309。现在的键盘挺多都是三模键盘，支持有线、无线和蓝牙三种连接方式，之前那个键盘在无线模式下按键响应有延迟，打游戏的时候明显能感受到，新的键盘就没有延迟，果然，一分钱一分货。我是挺喜欢这个新键盘，它的配色和我女朋友的一模一样，完全就是情侣款，如果重量可以再轻点就好了，现在每天上班都是拉练......
